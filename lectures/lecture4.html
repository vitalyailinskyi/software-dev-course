    VCS (II): SVN vs Git

    SVN - most popular centralized VCS. Client has only one piece of project history, he is currectly working with, called working copy. Every save or load opertaions requires connection to the server.

    Git - most popular decentralized VCS. Client has a clone of the whole repository. All changes can be done locally and synchonize with a server when it is needed.

    VCS (III): Git Terminology

    • Clone - copying whole project history from remote server to the client
    • Commit - saving changes were made on the client
    • Push - sending changes made and saved on the client to the server
    • Pull - loading changes from server to the client
    • Checkout - changing between various states of working copy on the client

      • Need help? http://git-scm.com/documentation

    Unit testing

    At a high-level, unit testing refers to the practice of testing certain functions and areas – or units – of our code. This gives us the ability to verify that our functions work as expected

    That is to say that for any function and given a set of inputs, we can determine if the function is returning the proper values and will gracefully handle failures during the course of execution should invalid input be provided.

    Example (I): Class to test

    						
public class Rectangle {
    private int leftHighX;
    private int leftHighY;
    private int rightLowX;
    private int rightLowY;
    
    public Rectangle(int leftHighX, int leftHighY, 
                     int rightLowX, int rightLowY) {}
    
    public int area() {
         return (rightLowX - leftHighX) * (leftHighY - rightLowY);
    }
}

    Example (II): Test case

    						
import static org.junit.Assert.assertEquals;

public class RectangleTest {
    @Test
    public int testArea() {
         Rectangle rec = new Rectangle(0, 0, 1, 1);
         assertEquals(1, rec.area());
    }

    @Test
    public int testArea_negativePoints() {
        Rectangle rec = new Rectangle(-1, -1, 2, 3);
        assertEquals(12, rec.area());
    }
}
